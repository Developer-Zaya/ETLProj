# Dockerized ETL Project with RabbitMQ and Apache Airflow

## Project Overview
This project demonstrates an ETL (Extract, Transform, Load) pipeline using RabbitMQ for message queuing and Apache Airflow for workflow orchestration, all containerized using Docker.

## Features
- **RabbitMQ:** For robust message queuing between ETL stages.
- **Apache Airflow:** For orchestrating the ETL workflow.
- **Docker:** Ensures consistent environment setup.

## Prerequisites
- Docker
- Docker Compose
